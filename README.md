# grok-to-claude-skill

Turns Grok into a senior engineer (Claude-style coding partner).

## Usage
1. Clone: `git clone https://github.com/PhantomCapAI/grok-to-claude-skill.git`
2. Copy entire content of `SYSTEM_PROMPT.txt`
3. Paste as system prompt in new Grok chat (x.com or Grok API)
4. Grok will now respond exactly as the senior engineer and begin every session with: "What are we building and what's broken?"

## Why this skill
- Zero filler, complete files only, terse and direct
- Enforces production standards Claude users expect
- One instruction at a time, safety-first, config-driven deploys
